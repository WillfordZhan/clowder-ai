# PPT 风格 Preset — 华为式专业汇报

> ppt-forge 华为式汇报风格完整定义。
> 来源：F144 华为密度实战 + 2026-05-28/29 imagegen 实战 + 2026-05-29 专业提示词规范。
> 核心设计理念：**数据为王，层级为纲，品牌红为魂。**

## 一、整体风格定位

企业级、专业、克制、数据导向。参考华为汇报型 PPT 的视觉语言：

- 信息传达优先，避免过度装饰
- 版式规整，层级清晰，适合战略汇报、经营分析、解决方案介绍
- 以品牌红为核心视觉锚点
- 大量使用白底、灰度分层、卡片式并列结构
- 重点数据使用红色放大
- 图表色彩遵循品牌配色规范，保持统一和克制

## 二、品牌主色应用规范

> 默认使用 PANTONE 185C 体系。实际使用时替换为目标公司品牌红。

### PANTONE 185C 系列

| 类型 | RGB | HEX | 用途 |
|------|-----|-----|------|
| **主色红** | 211/57/65 | `#D33941` | 关键数据、重点结论、强调标签、标题下划线 |
| 浅红变体 1 | 225/129/137 | `#E18189` | 次级强调、浅色背景、辅助标签 |
| 浅红变体 2 | 239/178/184 | `#EFB2B8` | 弱强调背景、图表浅色区块 |

### PANTONE 186C 系列（可选）

| 类型 | RGB | HEX | 用途 |
|------|-----|-----|------|
| 深红 | 211/56/89 | `#D33859` | 重要风险、战略主题、重点模块标题 |
| 玫瑰红 | 225/129/152 | `#E18198` | 次级强调、图表对比色 |
| 浅玫瑰 | 238/179/193 | `#EEB3C1` | 背景色、轻量提示区 |

### 主色使用原则

- 优先使用 `#D33941` 作为统一品牌红
- `#D33859` 可用于风险、挑战、战略重点等更强情绪表达
- **红色只用于真正重要的信息，不要整页大面积铺满**
- 推荐色彩比例：

| 元素 | 占比 |
|------|------|
| 白色/浅灰背景 | 70% |
| 深灰正文文字 | 20% |
| 品牌红强调色 | 8% |
| 辅助色 | 2% |

## 三、辅助色应用规范

辅助色主要用于图表、分类标签、业务板块区分，**不用于大面积装饰**。

| 色系 | RGB | HEX | 推荐用途 |
|------|-----|-----|---------|
| 洋红 | 196/0/84 | `#C40054` | 市场、用户、增长类数据 |
| 深红 | 127/0/1 | `#7F0001` | 风险、预警、负向指标 |
| 橙色 | 237/109/0 | `#ED6D00` | 机会、趋势、进展、节点提醒 |
| 黄色 | 253/211/81 | `#FDD351` | 提示、待关注事项、轻量预警 |
| 绿色 | 98/178/48 | `#62B230` | 正向增长、完成、健康状态 |
| 青色 | 48/181/197 | `#30B5C5` | 技术、平台、系统、创新模块 |
| 灰色 | 85/87/87 | `#555757` | 正文辅助、注释、图表坐标轴 |

## 四、灰度体系

灰色用于构建页面层级，保持专业感。

| 类型 | RGB | HEX | 用途 |
|------|-----|-----|------|
| 黑色 | 0/0/0 | `#000000` | 强标题、极少量重点文字 |
| 深灰 | 85/87/87 | `#555757` | 正文、分类标题、图表轴线 |
| 中灰 | 137/137/137 | `#898989` | 注释、来源、次级说明 |
| 浅灰 | 181/181/181 | `#B5B5B5` | 分割线、辅助边框 |
| 极浅灰 | 221/221/221 | `#DDDDDD` | 卡片边框、表格线 |
| 白色 | 255/255/255 | `#FFFFFF` | 主背景、卡片背景 |

**灰度使用原则**：
- 正文优先使用 `#555757` 或接近黑色的深灰
- 边框、分割线使用 `#DDDDDD`
- 注释、来源使用 `#898989`
- 背景以白色为主，可搭配极浅灰信息区块

## 五、字体规范

整体使用无衬线字体，保持商务和科技感。

- 中文字体：微软雅黑 / Microsoft YaHei
- 英文字体：Arial
- 数字字体：Arial

### 字号层级

| 层级 | 字号 | 样式 | 用途 |
|------|------|------|------|
| 页面主标题 | 28pt | 常规或中等字重 | 页面核心主题 |
| 分项标题 | 16-20pt | 加粗 | 模块标题、分类标题 |
| 关键数字 | 28-36pt | 加粗，Arial | KPI、营收、增长率 |
| 正文说明 | 11-14pt | 常规 | 业务说明、方法描述 |
| 标签文字 | 10-12pt | 常规或加粗 | 图例、分类、短标签 |
| 注释来源 | 6-9pt | 斜体，灰色 | 数据来源、备注说明 |

## 六、页面布局规范

采用华为式汇报页面三区结构：

### 1. 标题区（顶部）

- 位于页面顶部左侧
- 左边距约为页面宽度的 6%
- 标题字号 28pt
- 标题下方可使用品牌红细线强化页面识别

### 2. 内容区（中部）

- 使用卡片、表格、流程图、数据块组织信息
- 避免大段文字堆叠
- 每页控制在 3-6 个核心信息组

### 3. 注释区（底部）

- 使用 6-9pt 灰色斜体
- 包含数据来源、统计口径、版本说明

适合技术论证、性能提升证明、方案合理性论证。**不是工整分栏——是"以一个核心结论为中心，多种媒体素材从不同角度论证"。**

**视觉要求**：

- 面板类型混合使用（一页内至少 3 种不同类型）
- 不追求工整对称——以论点为中心的自然布局
- 底部红底白字总结条收束全页

**什么时候用**：当一页需要"用多种证据同时证明一个结论"时。如果只是展示数据（用 KPI 页）或对比方案（用 VS 页），不需要论证闭环页。

## 装饰元素层

华为 PPT 的品牌识别度很大程度来自**装饰元素的一致性**——每页都有统一的品牌装饰。

### 可选装饰元素

| 元素 | 描述 | 位置 | 何时用 |
|------|------|------|--------|
| **品牌飘带** | 红银色渐变飘带/波纹 | 页面底部或右下角 | 每页都加（品牌识别） |
| **科技光线** | 蓝白色放射光线 | 页面右侧或背景 | 技术/产品页 |
| **几何点阵** | 浅灰色点阵/网格 | 背景 | 数据/分析页 |
| **渐变色带** | 品牌红→白 渐变条 | 页面顶部标题下方 | 封面/总览页 |

### 使用原则

- **保持克制**：装饰只起品牌识别作用，不喧宾夺主
- **全套统一**：选定装饰元素后，每页都用同一种，不混搭
- **不遮挡内容**：装饰在最底层，不能覆盖文字/图表
- **红色占比计入 §二 的 8% 上限**：渐变色带建议高度 ≤3pt、宽度 ≤页宽 80%，飘带面积不超过页面 3%
- 低保真稿里这样标注：`底部装饰：品牌飘带（红银渐变）`

## 九、文字表达规范

### 标题表达

标题应直接表达结论，而不是泛泛描述。

| | 示例 |
|---|------|
| 推荐 | "2025 年核心业务保持稳健增长，收入结构持续优化" |
| 不推荐 | "业务情况介绍" |

### 正文表达

- 每个信息点控制在 15-30 字
- 避免长段落
- 使用短句、动宾结构
- 强调业务动作和数据结果

### 数据表达

关键数字品牌红，单位灰色：

```
4,175 亿元 ← 数字品牌红 28-36pt，单位灰色
 34.3% 同比增长 ← 同上
```

## 十、禁忌清单

生成 PPT 时**严禁**以下设计：

- 不使用渐变背景
- 不使用复杂阴影
- 不使用花纹、装饰性边框
- 不使用过多颜色（辅助色最多 5 种）
- 不使用艺术字体
- 不使用大段文字堆叠
- 不使用高饱和撞色
- 不使用复杂动效
- 不使用圆角（直角卡片）
- 不使用蓝色做主色

## 密度原则

华为密度 = **信息最大化 + 结构化承载**。

- 每页至少一个图表化元素（不能全是纯文字列表）
- 关键数据必须突出（品牌红大号）
- 用卡片/表格/矩阵组织信息，不用大段落
- 底部留一条红底白字总结条是华为的标志性手法
- 不怕密——怕密但乱。有结构的密是高级的

## Common Mistakes

| 错误 | 后果 | 修复 |
|------|------|------|
| 红色铺满整页 | 视觉疲劳，失去强调效果 | 红色只占 8%，用于真正重要的信息 |
| 用蓝色/橙色做主色 | 不是华为味 | 主色只用品牌红 |
| 圆角卡片 | 偏 Apple/互联网 | 直角 |
| 标题写"情况介绍" | 不传递信息 | 标题直接写结论 |
| 大段文字堆叠 | 受众看不下去 | 每点 15-30 字，用卡片/图表承载 |
| 数据没有来源 | 不专业 | 底部注释区标注数据来源 |
| 图表用高饱和多色 | 花哨不克制 | 按八、图表配色规则的优先级用色 |
| 渐变/阴影/3D | 廉价感 | 纯色 + 白底 |
